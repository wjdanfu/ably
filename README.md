## 시작하기
jdk가 설치가 안되어있으면 먼저 설치해주세요 (본 과제는 jdk-11 환경에서 만들어 졌습니다)

터미널을 열고 압축을 푼 폴더에서 다음 코드를 입력하여 실행시켜줍니다.
```
./gradlew clean build
java -jar build/libs/myably-0.0.1-SNAPSHOT.jar
```
